# Bookmark Workbench - Chrome拡張機能

Chromeのブックマークをツリー形式で直接編集できる拡張機能です。  
インポート/エクスポート不要で、変更は即座にChromeのブックマークに反映されます。

## 機能

- 📂 ブックマークをツリー形式で一覧表示（仮想スクロールで大量データも軽快）
- ✏️ 名前・URLの編集（右パネルの詳細画面 or 右クリックメニュー）
- ➕ ブックマーク・フォルダの新規作成
- 🗑 削除（フォルダごと一括削除も可能）
- 🔀 ドラッグ＆ドロップで並べ替え・フォルダ間移動
- ⬆⬇ 右クリックメニューから上下移動
- 🔍 名前・URLで検索
- 🔄 外部からのブックマーク変更も自動検知して再読み込み
- ⌨️ キーボードショートカット（Ctrl+F: 検索、Delete: 削除、F5: 再読み込み）

## インストール方法

### 手順1: ファイルを配置

以下のファイル構成でフォルダに保存されていることを確認してください。

```
bookmark-tool-extensions/
├── manifest.json
├── background.js
├── bookmarks.html
├── bookmarks.css
├── bookmarks.js
├── README.md
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

### 手順2: Chromeに読み込む

1. Chromeで `chrome://extensions` を開く
2. 右上の **「デベロッパーモード」** をオンにする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. このフォルダ（`bookmark-tool-extensions`）を選択

### 手順3: 使う

- ツールバーに表示される拡張アイコンをクリックすると、新しいタブでBookmark Workbenchが開きます

## 使い方

| 操作 | 方法 |
|------|------|
| ブックマークの詳細を見る | ツリーの項目をクリック |
| フォルダの展開/折りたたみ | フォルダ名 or ▸/▾ をクリック |
| 名前・URLを編集 | 右パネルで編集して「保存」ボタン |
| 並べ替え | 行をドラッグ＆ドロップ |
| 右クリックメニュー | 右クリック or 長押し |
| 検索 | 🔍ボタン or Ctrl+F |
| 新規追加 | 上部の「URL追加」「フォルダ追加」ボタン |
| 削除 | 右パネルの削除ボタン or 右クリック→削除 |

## 注意事項

- 「ブックマーク バー」「その他のブックマーク」などのChrome標準フォルダは削除できません
- 変更はすべてChromeのブックマークに直接反映されます（元に戻す機能はありません）
- この拡張機能はローカルでのみ動作し、外部サーバーへのデータ送信は一切行いません
